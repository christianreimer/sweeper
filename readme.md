# Mine Sweeper

Simple terminal version of the Mine Sweeper game.

```bash
$ ./sweeper.py --help
A simple terminal version of Mine Sweeper.

Usage: sweeper.py [--size=<s>] [--bombs=<b>]

Options:
  --size=<s>   Size of the board [default: 8]
  --bombs=<b>  Number of bombs on board [default: 12]
  --help       This message
```
